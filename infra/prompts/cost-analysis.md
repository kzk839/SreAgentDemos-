この環境の Azure コスト分析を実施してください。
レポートはすべて日本語で作成してください。

## 分析対象

サブスクリプション全体のコストについて、先月と今月を比較分析してください。

## 分析内容

### 1. 全体サマリー
- 先月の合計コスト（USD）
- 今月の現時点までのコスト（USD）
- 今月末の予測コスト（日割り計算）
- 前月比の増減率（%）

### 2. リソースグループ別コスト
- リソースグループごとの先月 vs 今月のコスト比較
- 最もコストが高いリソースグループの特定

### 3. サービス（メーター）別コスト
- サービスカテゴリ別の先月 vs 今月のコスト比較（例: Virtual Machines, Azure Firewall, SQL Database, Container Apps, Storage 等）
- コスト増減が大きいサービスを上位 5 つリストアップ

### 4. 変動要因の分析
コスト変動が大きいサービスについて、以下を調査してください:
- リソースの追加・削除があったか（Activity Log で確認）
- SKU やサイズの変更があったか
- 使用量の増加があったか（メトリクスで確認）
- 一時的なスパイクか恒常的な増加か

### 5. グラフ
- 日別コスト推移（先月 vs 今月の折れ線グラフ）
- サービス別コスト構成比（円グラフまたは棒グラフ）

## レポート形式

1. エグゼクティブサマリー（3 行以内で全体像）
2. コスト比較テーブル
3. グラフ
4. 変動要因の分析と詳細
5. コスト最適化の提案（あれば）
